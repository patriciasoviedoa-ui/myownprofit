Checklist para publicar:
[ ] Subir index.html, robots.txt, sitemap.xml, favicon.svg, site.webmanifest, og-image.svg.
[ ] Subir .htaccess si el hosting es Apache.
[ ] Subir _headers si usas Netlify o Cloudflare Pages.
[ ] NO subir ads.txt.template como ads.txt hasta tener ID real de AdSense.
[ ] Crear email real: contacto@myownprofit.com.
[ ] Verificar dominio en Google Search Console.
[ ] Enviar sitemap: https://myownprofit.com/sitemap.xml
[ ] Probar indexación con URL Inspection.
[ ] Crear Google Analytics y agregar el tag cuando tengas Measurement ID.
[ ] Solicitar AdSense solo cuando haya contenido suficiente, navegación clara, contacto y políticas visibles.

Importante:
Google recomienda crear contenido útil y pensado para personas, y AdSense exige cumplir políticas del programa. La web ya tiene estructura, pero el crecimiento dependerá de ampliar artículos, agregar ejemplos originales y ganar autoridad externa.
